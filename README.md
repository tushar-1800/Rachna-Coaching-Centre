# Rachna Coaching Centre Website

Kid-friendly, vibrant, single-page website for **Rachna Coaching Centre** (Classes **1 to 8**).

## Run locally

From this folder:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173` in your browser.

## Files

- `index.html` — page content
- `styles.css` — design/theme
- `script.js` — mobile menu toggle
- `assets/poster.png` — poster image
